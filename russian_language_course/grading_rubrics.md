# Рубрики оценивания

## 1. Мини-описание художественного объекта (5 баллов)
| Критерий | Баллы |
|----------|-------|
| Точность (соответствие объекту) | 1 |
| Наблюдательность (детали) | 1 |
| Уместность стиля | 1 |
| Средства связи / логика | 1 |
| Отсутствие речевых ошибок | 1 |

## 2. Документ (заявление / расписка) (5 баллов)
| Реквизиты полные | 2 |
| Ясность формулировок | 1 |
| Отсутствие лишней эмоциональности | 1 |
| Отсутствие ошибок | 1 |

## 3. Мини-доклад (10 предложений) (10 баллов)
| Структура (вступление / основная часть / вывод) | 2 |
| Логичность абзацев | 2 |
| Терминологическая точность | 2 |
| Уместность стиля | 2 |
| Языковая правильность | 2 |

## 4. Лингвостилистический анализ (8 баллов)
| Определение стиля | 1 |
| Типы речи | 1 |
| Тема / основная мысль | 2 |
| Лексика (характеристика) | 1 |
| Тропы / фигуры (названы и функция) | 2 |
| Общая оценка уместности | 1 |

## 5. Аннотация (3 балла)
| Сжато | 1 |
| Тема + цель отражены | 1 |
| Без оценочных лишних слов | 1 |

## 6. Тезисы выступления (5 баллов)
| Логичное упорядочивание | 1 |
| Четкость формулировок | 2 |
| Отсутствие повтора / тавтологии | 1 |
| Соответствие цели | 1 |

Используйте рубрики как чек-листы при самооценке.
