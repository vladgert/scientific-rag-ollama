# Занятие 3. Публицистический и художественный стили. Ораторская речь. Переработка текста. Стилистический анализ.

## 1. Публицистический стиль
Цель: воздействовать + информировать.  
Черты: оценочная лексика, общественно значимая проблематика, риторические вопросы, призыв.

## 2. Художественный стиль
Цель: создание образа, эстетическое воздействие.  
Средства: тропы (эпитет, метафора, сравнение, метонимия, гипербола), фигуры (повтор, градация, инверсия, риторический вопрос).

## 3. Сравнение (кратко)
| Признак | Публицистический | Художественный |
|---------|------------------|----------------|
| Цель | Убедить / актуализировать | Создать образ |
| Эмоциональность | Оценочная | Образно-эстетическая |
| Адресат | Массовый | Читатель (эстетическо�� восприятие) |
| Средства | Клише + яркие оценки | Индивидуальные авторские |

## 4. Ораторская речь (алгоритм)
1. Цель (информировать / мотивировать / доказать)  
2. Анализ аудитории  
3. Тезисы (3–5)  
4. Аргументы (факт, пример, логика, аналогия)  
5. Вступление (зацепка)  
6. Основная часть (блоки)  
7. Заключение (итог + призыв / перспектива)

Формулы начала:
- «Кто из вас задумывался о том, что…»
- «Сегодня мы сталкиваемся с..."
- «Попробуйте вообразить…»

## 5. Переработка текста
План — нумерованные пункты.  
Тезисы — компактные формулировки без деталей.  
Конспект — сжатый пересказ с логикой исходного текста.  
Аннотация — 2–4 предложения: тема, цель, объект, без оценки.  
Реферат — структурно: введение (актуальность), основная часть (ключевые положения), вывод (итог, оценка).

## 6. Лингвостилистический анализ (шаблон)
1. Тема  
2. Основная мысль  
3. Жанр / стиль  
4. Типы речи (преобладающий + элементы)  
5. Лексика (термины, оценка, эмоциональность)  
6. Тропы / фигуры (цель)  
7. Синтаксис (виды предложений, повторы)  
8. Композиция (как разворачивается)  
9. Оценка уместности средств

## 7. Типичные стилистические ошибки
- Смешение стилей («Уважаемые товарищи, эта штука реально классная!»)
- Плеоназм («перспективное будущее»)
- Канцеляризм в художественном описании
- Неверная метафора (логически не сочетается)
- Устаревшие штампы («в свете вышеизложенного» в публицистике без необходимости)
