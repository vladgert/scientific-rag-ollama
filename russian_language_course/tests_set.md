# Тестовые задания (вариативный банк)

## Блок 1. Термины (множ. выбор)
1. Связность текста обеспечивают:  
   a) тропы  
   b) лексические повторы  
   c) синонимические замены  
   d) риторические вопросы  
   Ответ: b, c

2. К официально-деловому стилю не относится:  
   a) доверенность  
   b) расписка  
   c) очерк  
   d) заявление  
   Ответ: c

3. Признак научного стиля:  
   a) просторечия  
   b) эмоционально-оценочные штампы  
   c) терминологическая точность  
   d) обилие междометий  
   Ответ: c

## Блок 2. Установите соответствие
A) Повествование  
B) Описание  
C) Рассуждение  

1. «Почему эта техника эффективна — попробуем объяснить.» → C  
2. «Он смешал краски и быстро нанёс фон.» → A  
3. «Тонкие холодные линии пересекали тёплую плоскость.» → B

## Блок 3. Исправление ошибок
Найдите и исправьте (по 1):  
1. «В данной работе прослеживается яркая выразительная экспрессия.»  
2. «Свет играет важную роль, потому что за счет света выявляется освещённость предметов.»  
3. «Этот метод конкретно и чётко показывает визуальное восприятие.»  

(Ожидаемое: убрать плеоназмы, тавтологии, канцеляризмы.)

## Блок 4. Определение стиля
Фрагмент:  
«Заявленная техника демонстрирует устойчивость к воздействию влаги и механическим повреждениям.»  
Ответ: официально-деловой / технический (обоснование: безличность, терминоподобные слова).

## Блок 5. Короткий анализ (открытый)
Дан фрагмент (преподаватель выбирает): определить:  
1) стиль, 2) типы речи, 3) 1 троп, 4) основная мысль.

## Блок 6. Аннотация
Текст (8–10 предложений) → написать аннотацию (2–3 предложения: тема + цель).

## Блок 7. Трансформация

## Общий совет по проверке
- Плеоназм: лишнее прилагательное, повтор значения.  
- Тавтология: повтор корня / одинаковых слов.  
- Смешение стилей: эмоционально-разговорное слово в формал��ном документе.  
- Неверная метафора: нелогичность (несовместимые сферы).  
- Канцеляризм: «осуществлять», «наличествует» — если не обязателен.
